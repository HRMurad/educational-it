###### 2. Write the `correct answer` from the following options and give an explanation (2-5 lines).

```javascript
function sum(a, b) {
  return a + b;
}

sum(1, "2");
```

- A: `NaN`
- B: `TypeError`
- C: `"12"`
- D: `3`

<details><summary><b>Answer</b></summary>
<p>

#### Answer: D

<i>
When I call sum(1, "2") in JavaScript, the string "2" is automatically converted to the number 2, and then the addition is performed, resulting in 1 + 2, which equals 3. JavaScript performs this type coercion when you use the + operator with a number and a string.</i>

</p>
</details>
